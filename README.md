# Sistema-de-Invent-rio-e-Validade-Python-CLI-
Sistema CLI em Python focado em controle de inventário e validade de produtos. O projeto demonstra organização modular, manipulação de dicionários e automação de processos como sorteio de datas de validade e verificação de descarte. Possui menus interativos para cadastro, pesquisa e fluxo de vendas.
